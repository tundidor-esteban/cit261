Manipulating CSS Class Properties Using JavaScript

CSS or "Cascading Style Sheets" is a standard to define the styles of the differents elements of the DOM. So, HTML is oriented to the structure and CSS is oriented to the visual definition. If the HTML is the wall, the CSS is the paint. Actually a very powerful paint because you will be able to move the windows positions just changing the definition.


If you add JS to the equation, you can literally make magic and dynamically change the elements and definitions. You have 2 options to manipulate the styles with JavaScript:

  • Setting the style directly to an element
  • Setting or removing the class attribute from an element

Setting the style directly to an element

Every HTML element has a style attribute, and it is an Object. You will be able to set or specify a CSS property and set its value.

let myElement = document.querySelector(".code");
myElement.style.backgroundColor = "#D93600";
myElement.style.fontSize = "1.0em";

You can identify one by one by ID with 

document.getElementById("myID")
or searching for it and changing all the elements with the same class name with 
document.querySelector(".myClassName")
or 
document.getElementsByClassName("myClassName")

Setting or removing the class attribute from an element

An easier and clearer way is just changing the class attribute from an element or a collection of elements. The only issue with this approach is that you need to pre define it, instead of compose it dinamically. 

let div = document.querySelector("#testDiv");
div.classList.add("classA"); // add class "classA"
div.classList.add("classB"); // add class "classB"
div.classList.add("classC"); // add class "classC"
alert(div.classList);

div.classList.remove("classC"); // remove class "classC"

alert(div.classList);
